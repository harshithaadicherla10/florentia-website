# 🌸 Florentia – Modern Florist Website

Florentia is a modern, elegant, and fully responsive florist website built using HTML and CSS.  
It showcases a premium floral brand with sections like Hero Video, About, Store, Sustainability, and Newsletter.

🔗 Live Demo: https://your-vercel-link.vercel.app

---

## 🚀 Features

- 🎥 Hero section with background video
- 🧭 Sticky navigation bar
- 🛍️ Store page with product grid layout
- 🔎 Search bar with icon inside input
- 🎨 Clean premium UI inspired by modern florist templates
- 📱 Fully responsive design
- ☁️ Deployed using Vercel

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Bootstrap Icons
- Vercel (Deployment)
- GitHub (Version Control)
